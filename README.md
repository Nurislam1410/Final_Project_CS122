# Final_Project_CS122
Title: TripSafe - AI Model reducing car crashes
Authors: Nurislam Saliev, Syed Zoraiz Sabeel
Description: An AI Model that is trained on real U.S. government-collected data. It tells the risk level(high/medium/low) of a car crash on selected roads/streets/areas. 
             Moreover, it also provides the causes of car crashes and how common they are. 
             The goal of this application is to make drivers aware of car crashes on the roads/streets they are driving on.
